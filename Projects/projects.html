Smart sort: waste classifier
A machine learning–based web app to classify waste as dry or wet using image inputs. Built as a collaborative project with my team, focusing on sustainability through AI.
Smart Sort is an AI-powered web application designed to classify waste as dry or wet using image inputs. Built as a team project, our goal was to promote sustainable waste management by automating the initial sorting process. We used a convolutional neural network (CNN) trained on a curated dataset (TrashNet + RealWaste) and implemented the model using TensorFlow.

The frontend, built with HTML, CSS, and JavaScript, allows users to upload waste images through an intuitive UI. Once submitted, the image is analyzed by the backend, and the classification result is displayed in real time. The model was integrated into the app using Flask for the backend and REST API calls.

As a team, we divided responsibilities across model development, data preprocessing, UI design, and deployment. Our collaborative workflow included version control through GitHub, and we followed agile practices to ensure weekly progress and testing.
